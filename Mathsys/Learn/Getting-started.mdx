## What is Mathsys?

**Mathsys** is a hybrid language that combines the clarity of mathematical notation with the power of computation. While LaTeX excels at typesetting mathematics, it cannot be reliably parsed or executed. Mathsys bridges this gap by providing a language that outputs beautiful LaTeX typesetting while remaining fully runnable and computable.

## Syntax

### Basic Arithmetic

Mathsys supports standard arithmetic operations:

<CodeGroup dropdown>
    ```mathsys ArithmeticOperations.msd
    1 + 2
    # Results in 3

    10 - 3
    # Results in 7

    4 * 5
    # Results in 20

    15 / 3
    # Results in 5
    ```
    ```tex ArithmeticOperations.ltx
    1+2\\ \\\text{Results in 3}\\ 10-3\\ \\\text{Results in 7}\\ 4\cdot 5\\ \\\text{Results in 20}\\ \frac{15}{3}\\ \\\text{Results in 5}
    ```
</CodeGroup>

### Variables

You can assign values to variables using `=`:

<CodeGroup dropdown>
    ```mathsys VariableAssignment.msd
    x = 5
    y = 3
    x + y
    # Results in 8
    ```
    ```tex VariableAssignment.ltx
    x=5\\ y=3\\ x+y\\ \\\text{Results in 8}
    ```
</CodeGroup>

Variables can be used in expressions just like numbers:

<CodeGroup dropdown>
    ```mathsys ExpressionsAndVariables.msd
    a = 2
    b = 3
    a * b + 1
    # Results in 7
    ```
    ```tex ExpressionsAndVariables.ltx
    a=2\\ b=3\\ a\cdot b+1\\ \\\text{Results in 7}
    ```
</CodeGroup>

## LaTeX Output

All Mathsys code is directly transpilable to LaTeX, producing beautiful mathematical typesetting:

<CodeGroup dropdown>
    ```mathsys TranspilationExample.msd
    2x^2^ + 3x + 1
    ```
    ```tex TranspilationExample.ltx
    2x^{2}+3x+1
    ```
</CodeGroup>

Transpiles to:

$$
2x^{2}+3x+1
$$

## Why Mathsys?

Traditional programming languages require you to translate mathematical concepts into their syntax. Mathsys lets you write mathematics naturally.

This makes Mathsys perfect for:
- Mathematical modeling
- Scientific computing
- Educational purposes
- Prototyping mathematical algorithms

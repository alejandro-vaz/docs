## Usage

Mathsys tensors (or simply, vectors) allow you to represent arrays of values.

```ms3
# empty vector
v = []

# one's vector
v = [1, 1]

# matrix
v = [[1, 0], [1, 2]]
# this matrix has two vectors (columns)
# which are [1, 0] and [1, 2]
# 1 1
# 0 2
```
